Introduction

This demonstration website was created for the coursework purposes, specifically for the 1st coursework assignment of SOFT20181: Internet Application Programming module, year 2019/2020.
Task assigned was to create a front end only static webpage for the fictional web shop company (named CWKS in my solution). Product catalogue is stored in an xml file and all non-static information’s are loaded from it using JSON.
Main idea and guideline when writing this code was ability to reuse most of the code later in the module, so code is written in a way that items like slideshow images, products names and description can be filled in automatically (something like a template). This resulted in ability to add more products, categories and therefore slideshows and product description pages by simply modifying XML file contents, without need to touch source code of the webpage.
All goals mentioned in the Assignment specification document and more are demonstrated in following html files:

  • index.html file, containing
    • Self-coded JavaScript slideshow, in which images are automatically loaded from items specified in xml file, sorted and added to slideshow
    • Content filling using xml file accomplished by JavaScript
      • Ability to see featured products on front page
      • Automatically filling slideshow mentioned above

  • products.html file, containing
    • Content filling using xml file accomplished by JavaScript
      • Filtering products by category using variables inside URL

  • product.html file, containing
    • Self-coded JavaScript slideshow, in which images are automatically loaded for given item specified in URL value
    • Content (item) description filling using product details for given item specified in URL value
      • Adding other items from that category in “Products you might find interesting area”

All html files mentioned above are including necessary CSS and JavaScript files to fulfil capabilities mentioned above. Layout of these pages is fully coded using CSS grid, providing full scalability on various devices with different screen size.
